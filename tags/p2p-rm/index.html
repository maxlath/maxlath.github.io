P2P Resources Management

a concept I'm pushing as a way around vendors monopoly on resources/products and transactions data, preventing VRM to emerge in any meaningful way: let's start to exchange data between peers the way we would like professional vendors where doing - with open data and libre software, for instance-, accept them back in the discussion only if they play with the same rules.
